# Simple HTML Project

This is a minimal static starter project created by your assistant.

Files:
- `index.html` — main page
- `styles.css` — basic styles
- `script.js` — small interactive script

How to open:
- Double-click `index.html` in File Explorer to open in your default browser.
- Or from PowerShell, run:

```powershell
Start-Process .\index.html
```

Serve via a simple HTTP server (optional):

```powershell
# Python 3
python -m http.server 8000; # then open http://localhost:8000
```

Next steps:
- Add more pages or components.
- Install Live Server extension in VS Code for automatic reloads.
- Initialize git if you want version control.
